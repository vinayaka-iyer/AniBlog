# Description

Aniblog is a simple MERN stack blog I made to review the fundamentals. This can be used as a template repository to bootstrap simple CRUD MERN applications.

# Features

-   Simple and minimalist design
-   View blogs from around the world
-   Users can register and login to become an author
-   Implemented secure JWT Authentication
-   User can create a blog and share it on the blogs page
-   The user can add a blog title, a summary, a cover image and the blog itself
-   The user may edit/delete the blog if they wish to do so
-   Blogs support markdown and images, stored in MongoDB

### How to Setup:

-   Step 1: Download the project zip and extract all files
-   Step 2: Open the root folder and two terminals
-   Step 3: Go to the client directory in the first terminal, write 'npm run dev'
-   Step 4: Go to the api directory in the second terminal, write 'npm run dev'
-   Step 5: Open browser and go to http://127.0.0.1:5173/ to see the project
